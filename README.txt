SFK Memories Story Maker

Purpose:
- Separate offline/local generator for SFK Memories story/post images.
- No Firebase, no login, no Google Drive, no Apps Script.
- Photos are loaded only inside the browser for PNG generation.

How to use:
1. Upload this folder to GitHub Pages, for example /sfk-memories-maker/.
2. Open index.html.
3. Add title, caption, date, posted by, and photos.
4. Choose FB/IG Story or Original Post.
5. Download PNG or use Share.

Files:
- index.html
- style.css
- app.js
- manifest.webmanifest
- sw.js
- app-icon.svg
- icons/icon-192.png
- icons/icon-maskable-512.png

This app does not touch your current SFK ClassBoard files unless you upload it over them.
